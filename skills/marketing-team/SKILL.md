# Marketing Team Skill Pack

## Description
Complete AI marketing team that runs campaigns end-to-end. Based on 9x's proven workflow.

## Agents Involved
- Researcher (competitor research)
- Data Analyst (campaign strategy)
- Copywriter (creative assets)
- Skill Architect (workflow orchestration)

## Workflow
1. **Researcher** scrapes Meta Ads Library → competitor analysis
2. **Data Analyst** visits your site → understands brand/style guide
3. **Copywriter** generates campaign proposal → landing pages, scripts
4. **Review & Approval** → draft mode for human review
5. **Publish** → to Meta Marketing API

## Usage
```
@marketing-team "Launch campaign for Q2 product release"
@marketing-team "Analyze competitor XYZ and propose counter-campaign"
```

## Configuration
- Meta Marketing API credentials
- Brand guidelines document
- Competitor list
- Budget parameters
