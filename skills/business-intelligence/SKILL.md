# Business Intelligence Skill Pack

## Description
AI advisory board that analyzes your business overnight and delivers actionable insights. Based on Matthew Berman's 8-expert pattern.

## Agents Involved
- Researcher (data gathering)
- Data Analyst (pattern recognition)
- Moris (simulation/debate mode)
- Dahlia (action item extraction)

## Workflow
1. **Researcher** aggregates data:
   - Revenue metrics
   - GitHub activity
   - Support tickets
   - Marketing performance
   - Calendar density
2. **Data Analyst** identifies patterns
3. **Moris** runs simulation:
   - Finance expert perspective
   - Marketing expert perspective
   - Operations perspective
   - Growth perspective
4. **Debate** → synthesize disagreements
5. **Output** → priority-ranked recommendations

## Usage
```
@business-intelligence "Run overnight analysis"
@business-intelligence "Focus: Q2 revenue issues"
```

## Configuration
- Data sources (Stripe, GitHub, HubSpot, etc.)
- Business goals
