# Personal OS Skill Pack

## Description
Complete personal operating system with AI team handling your life. Based on Brian Casel's setup.

## Agents Involved
- Moris (Orchestrator - strategic direction)
- Dahlia (Assistant - life admin)
- Copywriter (content creation)
- Pro Coder (development tasks)

## Daily Flow
- **6am**: Dahlia checks to-do list → prioritizes tasks
- **Morning**: Researcher prepares meeting briefs
- **Throughout**: Dahlia handles glue work, scheduling
- **Evening**: Copywriter captures content opportunities
- **Night**: Pro Coder works on backlog

## Features
- Telegram/Slack integration
- Meeting prep with LinkedIn research
- Daily morning brief with news + tasks
- Backlog automation
- Content capture

## Usage
```
@personal-os "Morning brief"
@personal-os "Prep for meeting with Acme Corp"
@personal-os "What's on my calendar today?"
```

## Configuration
- Google Calendar access
- LinkedIn credentials
- To-do app API (Things 3, Todoist, etc.)
- Dropbox/Google Drive sync
